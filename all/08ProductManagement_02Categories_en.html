Categories User Guide

Main Components

  • ID: Unique identifier for each category.
  • Brand: Represents the name of the brand.
  • Category: Description of the category.
  • Actions: Options to edit or delete an entry.

Usage Instructions

  • To add a new category, click on the button.
  • Use the search bar to filter categories by name.
  • Select the checkbox next to a category to perform bulk actions if necessary.
  • To edit a category, click the edit (pencil) icon in the Actions column.
  • To delete a category, click the delete (trash) icon in the Actions column.

Detailed Explanation of Features

The interface above displays a table listing all categories along with their attributes:

  • ID: Each category is assigned a unique ID for tracking.
  • Brand: Lists brands like ADIDAS and NIKE.
  • Category: Specifies the type of product, such as "Günlük Ayakkabı" (Daily Shoes) or "Spor Ayakkabı" (Sport Shoes).
  • Total: Displays the total number of categories listed.

Use Case Scenario

Imagine you are managing a product database and need to add a new category:

Step 1: Click the button.
Step 2: Fill in the required fields for brand and category.
Step 3: Click 'Save' to add the category to the list.
Step 4: Use the search function to quickly access specific categories.
