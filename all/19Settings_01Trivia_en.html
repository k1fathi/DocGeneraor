Trivia Management User Guide

Main Components

  • Trivia Name: The name of the trivia game.
  • Trivia Start Date: The date and time when the trivia game begins.
  • Trivia End Date: The date and time when the trivia game ends.
  • Trivia Active Status: Indicates if the trivia game is currently active.
  • Number of Questions: The total count of questions in the trivia game.
  • Actions: Options to edit or delete the trivia game.

Usage Instructions

To manage your trivia games, follow these steps:
  • Use the Word Search field to find specific trivia.
  • Select the desired trivia game from the list shown.
  • Use the Add button to create a new trivia game.
  • To edit or delete an existing trivia game, use the respective icons in the Actions column.

Detailed Explanation of Features

  • Word Search: This search bar helps you quickly find a trivia game by entering keywords.
  • Filter Options: Use the drop-down menu to filter trivia by active status or other parameters.
  • Add Button: Clicking this allows you to initiate the process of creating a new trivia game.
  • Pagination: Navigate through multiple pages of trivia using the page numbers at the bottom of the list.

Use Case Scenario

Scenario: A user wants to create a new trivia game.
  • The user clicks on the Add button.
  • They fill in the trivia name, start date, end date, and set the number of questions.
  • Once completed, they save the new trivia game.
