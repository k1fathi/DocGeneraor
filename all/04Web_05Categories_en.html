Categories User Guide

Main Components

  • Word Search Input: A field for entering keywords to filter categories.
  • Filter Options: Dropdowns for filtering active and deleted categories.
  • Categories List: Displays the available categories along with their actions.
  • Add Button: A button to add a new category.
  • Actions Column: Contains buttons for editing and deleting categories.

Usage Instructions

  • To search for a category, type a keyword in the Word Search field.
  • Select filter options to show only active or deleted categories.
  • Click the Add button to create a new category.
  • To edit an existing category, click the edit icon in the Actions column.
  • To delete a category, click the delete icon in the Actions column.

Detailed Explanation of Features

Word Search

The Word Search allows you to quickly find a category by typing keywords related to it.

Filter Options

Two dropdown filters help you manage the visibility of categories. Select the Active or Deleted options to customize your view.

Categories List

This section displays all categories, their record numbers, and available actions like edit and delete.

Use Case Scenario

Assume you want to find a category called "Kravats". You would:

1. Enter "Kravats" in the Word Search field. 2. If necessary, adjust the Filter Active option to show only active categories. 3. Review the results in the categories list. 4. If you want to edit, click the edit icon next to "Kravats".
