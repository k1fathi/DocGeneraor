Segments Management User Guide

Main Components

  • Segment List: Displays all segments with their respective details.
  • Filters: Options to filter segments based on various criteria.
  • Add Button: Allows users to create new segments.
  • Action Icons: Provides options to edit or delete segments.

Usage Instructions

  • Searching Segments: Use the search bar at the top to find a specific segment by name.
  • Applying Filters: Select filters to view only active or deleted segments.
  • Adding a New Segment: Click the button to create a new segment.
  • Editing a Segment: Click the edit icon next to a segment to modify its details.
  • Deleting a Segment: Click the delete icon to remove a segment from the list.

Detailed Explanation of Features

  • Segment No: A unique identifier for each segment.
  • Segment Name: The name of the segment visible to users.
  • Required Points: Points a customer needs to belong to the segment.
  • Reward Points: Points that are rewarded to customers in the segment.
  • Customer Count: The number of customers currently in the segment.

Use Case Scenario

A user wants to create a new segment named "Loyal Customers". They click the button, enter the necessary details including required points of 500 and reward points of 50. Once completed, they click on the save option to finalize the creation of the new segment.

2. Enter Segment Name
