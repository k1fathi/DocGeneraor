Notifications User Guide

Main Components

  • Record No. - Unique identifier for each notification record.
  • Scenario Name - Name of the scenario associated with the notification.
  • Customer No. - Unique identifier for the customer.
  • Message Details - Description or message content related to the notification.
  • Campaign Detail - Specific details regarding the campaign linked with the notification.
  • Push Subscriber ID - Identifier for the subscriber who received the push notification.

Usage Instructions

To view notifications:
  • Navigate to the Notifications section.
  • Use the Search field to filter notifications by keywords.
  • Utilize the Filter options to narrow down by User ID, Campaigns, or Scenarios.

Detailed Explanation of Features

  • User ID Filter: Allows filtering notifications by specific users.
  • Campaign Filter: Select campaigns to view notifications associated with those campaigns.
  • Scenario Filter: Choose scenarios to see relevant notifications.

Use Case Scenario

A user wants to check notifications related to a specific campaign:
  • Access the Notifications interface.
  • In the Campaigns dropdown, select the desired campaign.
  • View the filtered notifications that appear based on the selected campaign.
