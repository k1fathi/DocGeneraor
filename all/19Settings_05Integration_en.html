User Guide for Integrations

Main Components

  • Connection Type: Type of connection utilized (e.g., SMS, Push, Email).
  • Connection URL: Endpoint for the respective API for integration.
  • Username & Password: Credentials for authentication.
  • Created At: Timestamp indicating when the integration was created.

Usage Instructions

Follow these steps to configure your integrations:
  • Click on the Add button located at the top right corner.
  • Select the appropriate Connection Type from the dropdown menu.
  • Enter the required details: URL, Username, Password, etc.
  • Once filled, click the Save button to store your settings.

Detailed Explanation of Features

  • SMS Integration: Allows sending SMS through various providers.
  • Push Notifications: Enables sending push notifications via configured URLs.
  • Email Integration: Facilitates sending emails via specified SMTP settings.

Use Case Scenario

A typical user interaction would be:
  • Navigate to the Integrations section from the main dashboard.
  • Type in the name of the service you want to add in the Word Search field.
  • Select the Connection Type to filter relevant integrations.
  • Review existing integrations and click Edit or Delete as needed to manage them.
