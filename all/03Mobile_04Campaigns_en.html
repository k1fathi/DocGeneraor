Campaigns User Guide

User Guide: Campaigns Interface

Main Components

  • Campaigns Header: Main title for the campaigns section.
  • Word Search: Input box for searching specific campaigns.
  • Filters: Options to filter active or deleted campaigns.
  • Add Button: Button to create a new campaign.
  • Campaign List: Display of recorded campaigns with actions.
  • Actions: Icons for editing or deleting campaigns.

Usage Instructions

To use the Campaigns interface, follow these steps:

  • Type a keyword into the Word Search field to find specific campaigns.
  • Select filter options to view either Active or Deleted campaigns.
  • Click the button to create a new campaign.
  • Select the desired action (edit or delete) from the campaign list.

Detailed Explanation of Features

  • Word Search: Helps to quickly locate campaigns based on keywords.
  • Filters: Adjust visibility of campaigns by their status. Options include:
    • Only Active
    • Only Not Deleted
  • Add New Campaign: Use the button to navigate to the creation interface.
  • Campaign List: Displays the campaigns complete with their unique Record No. and actions.
