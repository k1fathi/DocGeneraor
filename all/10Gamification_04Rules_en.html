Rules Management Interface User Guide

Main Components

  • ID: Unique identifier for each rule.
  • Rule Name: The name of the rule.
  • Rule Description: Explanation of what the rule does.
  • Rule Type: Type of rule (e.g., Gain XP).
  • Expected Event String: Event that triggers the rule.
  • Event Count: Number of times the event has occurred.
  • Reward: Points or rewards granted for fulfilling the rule.
  • Actions: Edit or delete the existing rules.

Usage Instructions

  • To add a new rule, click on the button.
  • Use the Word Search to filter rules based on the name or description.
  • Select the Filter Deleted dropdown for viewing specific rules.
  • Review the Actions column for options to edit or delete a rule.

Detailed Explanation of Features

ID: Each rule displays a unique ID to help identify and manage it easily.
Rule Name: Clearly indicates what action the rule pertains to, making it easy to understand its purpose.
Rule Description: Provides a detailed explanation of the rule's function, which is crucial for users to comprehend how it impacts the gamification system.
