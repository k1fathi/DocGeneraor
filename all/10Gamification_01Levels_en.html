User Guide for Levels Interface

Main Components

  • ID
  • Name
  • Platform
  • Description
  • Running Status
  • Actions

Usage Instructions

To add a new level:
  • Click on the button.
  • Fill in the required information in the form that appears.
  • Submit the form to create the new level.

Detailed Explanation of Features

  • ID: Unique identifier for each level.
  • Name: The name assigned to the level.
  • Platform: Indicates if it's for Web, Mobile, or No Platform.
  • Description: Brief description of the level.
  • Running Status: Displays whether the level is currently Running or Stopped.
  • Actions: Includes buttons for viewing, editing, and deleting each level.

Use Case Scenario

A user needs to manage the levels within a gamification system. To do this, they:
  • Access the Levels interface from the main menu under Gamification.
  • Use the search bar to find a specific level by its name or ID.
  • Select a level to view its details using the view button.
  • Edit the level's information as needed using the edit button.
  • Delete a level that is no longer needed using the delete button.
