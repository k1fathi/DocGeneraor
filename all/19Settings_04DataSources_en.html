Data Sources User Guide

Main Components

  • Data Source List: Displayed items with details such as name, type, and status.
  • Actions Menu: Buttons for editing and controlling data sources.
  • Search Box: Field for searching through the data sources.

Usage Instructions

To manage your data sources, follow these steps:
  • Use the Search Box to find specific data sources based on name or criteria.
  • Select a data source from the list to view its details.
  • Click on the Edit button to make changes to the selected data source.
  • Use the + Add button to create new data sources.
  • Confirm your changes with the Save button.

Detailed Explanation of Features

  • Data Source Name: The label indicating what the data source is for, ensuring clarity.
  • Connector Type: Defines the mechanism used to retrieve data, such as API, database, etc.
  • Status: Indicates whether the data source is active, inactive, or has issues.
