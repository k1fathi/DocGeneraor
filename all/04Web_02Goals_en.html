User Guide for Goals Interface

Main Components

  • Record No.: Unique identifier for each goal entry.
  • Goal: Descriptive text representing the goal.
  • Word Search: A search bar to filter goals by keywords.
  • Filter Active: Options to display only active goals.
  • Filter Deleted: Options to manage viewing deleted goals.
  • Add Button: A button to add new goals to the list.

Usage Instructions

  • To add a new goal, click the button.
  • To search for specific goals, enter keywords in the Word Search field.
  • Select filtering options for active and deleted goals to customize your view.
  • Click on a goal entry to view further options for editing or deleting the goal.

Detailed Explanation of Features

Filters

  • Word Search: Type in keywords to find specific goals.
  • Filter Active: Choose "Only Active" to view current goals.
  • Filter Deleted: Choose "Only Not Deleted" to exclude deleted entries.
