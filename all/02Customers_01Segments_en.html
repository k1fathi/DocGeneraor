Segments User Guide

Main Components

  • Segment No: Identifier for each segment.
  • Segment Name: Name associated with the segment.
  • Required Points: Points needed for participation.
  • Reward Point: Points earned upon completion.
  • Customer Count: Number of customers in the segment.
  • Filters: Used for refining the displayed segments.

Usage Instructions

Adding a New Segment:

  1. Click the button.
  2. Fill in the necessary details in the provided form.
  3. Submit the form to create a new segment.

Detailed Explanation of Features

  • Word Search: A search field to quickly locate specific segments based on name.
  • Filter Active: Options to display only active or inactive segments.
  • Filter Deleted: Options to show segments that are deleted or not deleted.

Use Case Scenario

A user wants to find specific segments that have a reward point greater than 10. They can utilize the Word Search field to input keywords related to the segments. If the user wants to filter further, they can select Only Active from the Filter Active dropdown menu to see only segments currently in use.
