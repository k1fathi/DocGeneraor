User Guide for Categories Interface

Main Components

  • Word Search: A text input field to search for specific categories.
  • Categories Dropdown: A selection menu to filter categories.
  • Filter Deleted: A toggle to show or hide deleted categories.
  • Add Button: A button to add new categories to the list.
  • Data Table: Displays a list of categories with various attributes.

Usage Instructions

To manage categories, follow these steps:
  1. Enter a keyword in the Word Search field to filter the category list.
  2. Select a category type from the Categories dropdown menu.
  3. Use the Filter Deleted option to show or hide deleted items.
  4. Click on the button to create a new category.

Detailed Explanation of Features

  • ID: Unique identifier for each category.
  • Group Name: The name associated with a grouping of categories.
  • Name: The specific title of the category.
  • Create Time: Timestamp indicating when the category was created.
  • Update Time: Timestamp for the last update made to the category.
  • Delete Time: Shows when the category was deleted, if applicable.
  • Actions: Icons to edit or delete the category.

Use Case Scenario

When a user wants to add a new category:
  1. The user clicks the button.
  2. A form will appear requiring details for the new category.
  3. After filling in the required fields, the user submits the form.
  4. The new category is then listed in the data table.
  5. If the user wishes to delete an existing category, they can click the delete icon in the Actions
