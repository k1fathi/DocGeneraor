Brands Management User Guide

Main Components

  • ID: Unique identifier for each brand.
  • Brand: Name of the brand displayed in the list.
  • Actions: Options to edit or delete a brand.
  • Add Button: Access to add a new brand.
  • Search Bar: Input for searching brands in the list.
  • Pagination: Controls to navigate through multiple pages of brands.

Usage Instructions

To manage brands, follow these steps:
  • View Brands: The table displays all existing brands with IDs.
  • Add New Brand: Click the button.
  • Edit Brand: Click the edit icon next to the brand you wish to edit.
  • Delete Brand: Click the trash bin icon next to the brand to remove it.
  • Search Brands: Type into the search bar to filter brands.
