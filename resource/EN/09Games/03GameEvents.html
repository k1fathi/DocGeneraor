Game Events User Guide

Main Components

  • Select Date: Input field to choose the date for event filtering.
  • Games: Dropdown menu for selecting a specific game.
  • Filter Button: Button to apply the selected filters.
  • Event Table: Displays game events based on selected filters.

Usage Instructions

  1. Choose the desired date from the Select Date field.
  2. Select a game from the Games dropdown menu.
  3. Click the button to view events.
  4. If no events are found, a message will indicate that no game events are available.

Detailed Explanation of Features

  • Select Date: This field requires a date in the format mm/dd/yyyy. It allows users to filter events by date.
  • Games: This dropdown allows users to choose from a list of games. Selecting a game will impact the events displayed.
  • Event Table: The table shows the filtered results. If no events match the criteria, the message "Game event not found." will appear.

Use Case Scenario

Once a user logs into the system, they navigate to the "Game Events" page. They enter "10/31/2023" in the Select Date field and choose "Game A" from the dropdown. After clicking the , they see a list of events related to "Game A" for the selected date. If there are no events for that date and game, they will be informed that no game events are found.
