User Guide for Goals Interface

Main Components

  • Word Search: A search field to filter goals by keywords.
  • Filter Deleted: Options to filter out deleted items.
  • Add Button: A button to create new goals.
  • Table Header: Displays columns for ID, Name, Description, Journey, Rule, Segments, Start Date, End Date, and Actions.
  • Pagination: Navigation for browsing multiple pages of goals.

Usage Instructions

To manage goals in the interface, follow these steps:
  • Use the Word Search field to enter keywords that match the goals you're looking for.
  • Select options in the Filter Deleted dropdown to show only active goals.
  • Click the button to create a new goal.

Detailed Explanation of Features

  • ID: A unique identifier for each goal, sortable in ascending or descending order.
  • Name: The title of the goal, providing a clear reference.
  • Description: A brief summary of the goal's purpose and scope.
  • Journey: Specifies the progress or steps involved in achieving the goal.
  • Rule: Outlines the criteria for success related to the goal.
  • Segments: Different phases or parts of the goal.
  • Start Date: When the goal was initiated.
  • End Date: The completion date of the goal.
  • Actions: Availability of options such as edit or delete for specific goals.

Use Case Scenario

A user wants to create a new goal to increase user engagement:
  1. They type "Engagement" in the Word Search to see related goals.
  2. They set the Filter Deleted option to view active goals
