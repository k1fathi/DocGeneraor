SMS Interface User Guide

Main Components

  • SMS Services: Dropdown menu to select the SMS service.
  • Name Surname: Input field for the recipient's name and surname.
  • GSM No: Input field for the recipient's GSM number.
  • Content: Text area for typing the SMS message.
  • Send Button: Button to submit the SMS.

Usage Instructions

Follow these steps to send an SMS:
  • 1. Select an SMS service from the SMS Services dropdown.
  • 2. Enter the recipient's Name Surname in the corresponding field.
  • 3. Input the recipient's GSM No.
  • 4. Type the message content in the Content text area.
  • 5. Press the button to send the SMS.

Detailed Explanation of Features

  • SMS Services: Allows users to choose from various SMS service providers.
  • Name Surname: Helps identify the recipient of the SMS.
  • GSM No: Mandatory field to ensure the message is sent to the correct number.
  • Content: Can include any message, limited by character constraints from the service.
  • Send Button: Submits the entered information for processing the SMS.

Use Case Scenario

For instance, you want to notify a customer about a promotional offer:
  • Step 1: From the SMS Services, select "Promotional SMS."
  • Step 2: Enter "John Doe" in the Name Surname field.
  • Step 3: Input "123456789" in the GSM No field.
