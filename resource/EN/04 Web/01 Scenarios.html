User Guide for Scenarios Management

Main Components

  • Record No. - Unique identifier for each scenario.
  • Scenario Name - The name of the scenario.
  • Campaign - Associated campaign for the scenario.
  • Action Type - Type of action performed (e.g. Category View).
  • Action Selection - Specific selection within the action type.
  • Action Total - Total number of actions recorded.
  • Award Point - Points awarded for completing the action.
  • Event Message - Message displayed upon event completion.
  • Actions - Buttons for editing or deleting the scenario.

Usage Instructions

To Add a New Scenario:
  • Click the button.
  • Fill in the required fields for the new scenario.
  • Click Save to create the new scenario.

Detailed Explanation of Features

  • Word Search: Use this field to search scenarios by name.
  • Filter Active: Toggle between active and inactive scenarios.
  • Filter Deleted: View deleted scenarios or only active ones.
  • Search: Utilize this input to find specific records quickly.

Use Case Scenario

Example Scenario:
  • A user wants to review active scenarios related to "Electronics".
  • They enter "Electronics" in the Word Search field.
  • They set "Only Active" in the Filter Active dropdown.
  • The system displays relevant scenarios, allowing the user to select actions from the Actions column.
