Scenarios Management Guide

Header Section

  • Breadcrumb Navigation (Top Right): Displays the path Mobile > Scenarios.
    Usage: Click on "Mobile" to return to the main screen, or "Scenarios" to reload the current page.

Filter and Search Controls

  • Word Search: Searches for scenarios using specific keywords.
    Example: Type maxis to find a matching scenario.
  • Filter Active: Filters scenarios by their active status.
    • Only Active: Displays only active scenarios.
    • All: Displays both active and inactive scenarios.
  • Filter Deleted: Filters based on whether scenarios are deleted or not.
    • Only Not Deleted: Shows scenarios that are not deleted.
    • All: Shows both deleted and non-deleted scenarios.
  • Filter Button: Applies the selected filters.
    Usage: Select Only Active and Only Not Deleted, then click to apply filters.

Add Button (+ Add)

  • Adds a new scenario to the system.
    Usage: Click to open the scenario creation form.

Search Field (Bottom Right)

  • Searches for a specific scenario by name or record number.
    Example: Type okula dönüş to locate the matching scenario.

Scenario List Table

  • Record No.: A unique identifier for each scenario.
  • Scenario Name: Displays the scenario’s name.
    Example: The scenario maxis has Record No. 14.
  • Campaign: Shows the related campaign name.
  • Action Type: Describes the type of action in the scenario.
  • Action Selection: Defines the specific selection criteria for actions.
  • Action Total: Displays the total number of actions required.
  • Award Point: Points awarded for completing the scenario.
  • Event Message: Displays a custom message related to the event.
    Example: Scenario 12 shows the message (x) score submitted..
  • Actions: Provides options to manage scenarios:
    • Edit (Pencil Icon): Opens the scenario for editing.
    • Hide/Show (Eye Icon): Toggles the visibility of the scenario.
    • Delete (Trash Icon): Marks the scenario as deleted.
    Usage: Click the pencil icon to modify the maxis scenario.
