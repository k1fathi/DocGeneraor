Dynamic Data User Guide

Main Components

  • Dynamic Data Header: Displays the page title and current section.
  • Search Filters: Input fields for Word Search, User ID, and Date selection.
  • Data Table: Contains filtered results with various columns including Record ID and User ID.
  • Export Excel Button: Allows users to export data results to an Excel file.

Usage Instructions

Step 1: Use the Word Search field to enter keywords to filter the data.
Step 2: Enter the specific User ID you want to filter results by.
Step 3: Select the desired date from the Select Date calendar option.
Step 4: Click the button to apply your filters.

Detailed Explanation of Features

  • Word Search: A text input that helps to filter results based on specific keywords.
  • User ID: Input field to filter records related to specific users.
  • Select Date: Date picker to select the relevant date for filtering the data set.
  • Data Table Columns: Includes columns like Record ID, Action Type, Email, and other relevant data for each entry.
  • Export Excel: A feature to download the displayed filtered records in Excel format.

Use Case Scenario

A user logs into the system and wants to review actions taken by a specific user. They enter the User ID into the corresponding field, specify a date of interest, and input a keyword in the Word Search box. After clicking on the button, the data table updates to display only the relevant records matching their criteria.
